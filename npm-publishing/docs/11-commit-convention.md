# Commit Message Reference

## Format

```
<type>(<scope>): <subject>

<body>
```

## Quick Commands

```bash
# Use the interactive commit helper
npm run commit

# Manual commit (will be validated)
git commit -m "feat(llm-router): add new feature"
```

## Rules

✅ **DO:**

- Use present tense: "add" not "added"
- Keep subject lowercase
- Keep subject under 100 characters
- Be specific and descriptive

❌ **DON'T:**

- End subject with a period
- Be vague ("fix stuff", "update code")
